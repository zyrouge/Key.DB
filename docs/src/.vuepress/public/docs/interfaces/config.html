Interface Config

KeyDB Configuration (Common for all Dialects)

Example:

SQLite

const config = {
     dialect: "sqlite",
     storage: "./database.sqlite"
}

Postgres

const config = {
     dialect: "postgres",
     username: "someuser",
     password: "youshallnotpass",
     host: "localhost",
     port: "8080"
}

MongoDB

const config = {
     dialect: "mongodb",
     uri: "mongodb://mongodb0.example.com:27017"
}
