KeyDB

A simple promise-base key-value database ORM

Installation

 
$ npm install key.db

and also a Dialect

$ npm i pg pg-hstore # Postgres 
$ npm i mysql2 # MySQL
$ npm i mariadb # MariaDB
$ npm i sqlite3 # SQLite
$ npm i tedious # Microsoft SQL Server
